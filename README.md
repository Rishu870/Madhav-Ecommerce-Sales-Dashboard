# Madhav-Ecommerce-Sales-Dashboard
Power BI Sales Dashboard using DAX and Business Insights
Project Overview

This project presents a Sales Performance Dashboard built using Power BI and DAX for Madhav Ecommerce.
The dashboard provides insights into sales performance, profit trends, category contribution, and state-wise analysis.

**Business Objective**

The goal of this dashboard is to:

Analyze total sales and profit

Compare quarterly performance (Qtr1‚ÄìQtr4)

Evaluate state-wise profitability

Identify top-performing sub-categories

Monitor monthly profit trends

Understand category-wise quantity distribution
